## 🚀 未來優化改良與可開發功能

### 🟢 第一優先 ✅ 已全數完成（v1.2.0）

### 🟡 第二優先（中等工程，部分需後端或素材）

| 項目 | 說明 | 需求 | 工時 |
|---|---|---|---|
| **家長線上回條 / 出席登記** | 估算出席人數、製作座位卡 | Google 表單嵌入（零後端）或 Firebase | 中 |
| **交通與停車資訊** | Google Maps 連結／嵌入 + 停車動線 | 你提供停車與交通資訊 | 中 |
| **典禮直播頁內嵌** | 直接在站內看直播（賽前倒數→直播中嵌入播放器→賽後回放），不必跳離網站 | 純前端（YouTube iframe）+ 日期感知 | 中 |
| **祝福留言牆** | 學生／家長線上留言祝福、即時顯示 | 需後端 + **審核機制** → Firebase Firestore + rules | 較大 |
| **畢業歌〈啟程〉歌詞 + 影片 / BGM** | 歌詞區塊、影片嵌入、可靜音 BGM | 素材；BGM 走 `pixabay-audio` 技能 | 中 |
| **校園巡禮互動地圖** | 11 站路線畫成校園平面圖動線 | 校園平面圖素材 | 中–較大 |
| **深色模式** | `prefers-color-scheme` 自動切換 | 調一套深色配色 | 中 |

### 🔵 進階／延伸

- **內容資料化（JSON 驅動）+ 簡易後台**：流程／文案抽成資料檔，非工程同事也能改。
- **畢業生風采 / 班級榮譽榜**：照片牆、得獎名單。⚠️ 務必先取得家長同意（個資 / 肖像權）。
- **PWA 完整離線 + 安裝引導**：真正「加到主畫面」當 App。
- **多語言（中／英）**：新住民家庭友善。
- **流量分析（Plausible / GA4）**：看流量與來源（注意隱私揭露）。
- **典禮後封存紀念頁**：典禮後自動轉成回顧頁（影片、相簿、花絮）。
- **OG 圖日期感知**：典禮後分享卡改成「感謝 / 回顧」版。
