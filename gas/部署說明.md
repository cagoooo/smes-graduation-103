# 家長線上回條 — 後端部署說明（約 5 分鐘）

回條資料會寫進你自己的 Google 試算表，老師打開試算表就能看。**請用學校帳號 `ipad@mail2.smes.tyc.edu.tw` 操作**。

## 步驟

1. **建立試算表**
   用 `ipad@` 開一份新的 Google 試算表，命名為「第103屆畢業典禮-家長回條」。

2. **開啟 Apps Script**
   在試算表上方選單：**擴充功能 → Apps Script**。

3. **貼上後端程式**
   - 把編輯器裡預設的 `Code.gs` 內容**全部刪掉**，貼上本資料夾 [`Code.gs`](Code.gs) 的內容。
   - （建議）左側齒輪 ⚙ **專案設定 → 勾選「在編輯器顯示 appsscript.json」**，回到編輯器把 `appsscript.json` 內容換成本資料夾的版本。
   - 存檔（💾）。

4. **部署為網頁應用程式**
   右上 **部署 → 新增部署 → 類型選「網頁應用程式」**：
   - 說明：`回條 v1`
   - 執行身分：**我（ipad@…）**
   - 具有存取權的使用者：**任何人**
   - 按 **部署**。

5. **授權（第一次必做）**
   部署時會跳授權 → 選 `ipad@` 帳號 → 「進階」→「前往專案（不安全）」→「允許」。
   （或回編輯器，函數下拉選 `authorize` → 按「執行」走完同樣授權。）

6. **複製網址給阿凱**
   部署完成後會顯示「網頁應用程式」網址，長得像：
   `https://script.google.com/macros/s/AKfyc..../exec`
   **把這串 /exec 網址貼給阿凱**（或自行填入 `script.js` 裡的 `RSVP_ENDPOINT`），表單就會上線。

## 完成後

- 家長在網站「線上回條」送出 → 試算表「回條」分頁自動新增一列（送出時間／班級／姓名／人數／悄悄話）。
- 要看回覆，直接打開那份 Google 試算表即可。
- 日後若想改部署，**請用「管理部署 → 編輯（鉛筆）→ 版本：新版本」**更新，網址不變。

> 小提醒：測試時用無痕視窗開網站，避免多個 Google 帳號登入造成的權限混亂。
